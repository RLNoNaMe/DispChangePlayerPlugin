# DispChangePlayerPlugin
ロケットリーグのMODの1つ、BakkesModのプラグインです。

## プラグインのインストール方法
1. BakkesModを起動し[File]->[Open BakkesMod folder]でフォルダを開く
2. pluginsフォルダを開き、DispChangePlayerPlugin.dllファイルを設置する
3. settingsフォルダを開き、dispchangeplayerplugin.setファイルを設置する
4. 手順1で開いたフォルダまで戻る
5. cfgフォルダを開き、plugins.cfgをメモ帳で開き「plugin load dispchangeplayerplugin」を追記する
6. ロケットリーグを起動する

## プラグインの操作方法
### プラグインの操作画面の表示
1. F2キーを押し、BakkesModのメニュー画面を開く
2. pluginsタブを押し、左のサイドメニューから「DispChangePlayerPlugin」を選択
